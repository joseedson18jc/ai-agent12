# /content-create
Usage: /content-create <topic> [--type article|blog|social|newsletter] [--audience <target>]
Create content through a multi-stage AI pipeline: research, write, edit, SEO optimize.
## Steps
1. Research the topic: gather key points, facts, and angles
2. Write draft content matching the type and audience
3. Edit for grammar, flow, clarity, and tone
4. SEO optimize: generate title, meta description, keywords, readability score
5. Output the final content with metadata
