# PDF & Excel Report Generation
Generate professional PDF and Excel reports programmatically with charts and formatting.
## Instructions
- Use PDFKit (Node) or jsPDF (browser) for PDF generation
- Use ExcelJS (Node) or SheetJS (browser) for Excel/XLSX generation
- Structure reports with: header, summary section, data tables, charts, footer
- For invoices: include line items with quantities, unit prices, subtotals, tax, and total
- For financial reports: include KPI summary, trend charts, and detailed breakdowns
- Apply consistent branding: company logo, colors, fonts
- Support multiple output formats from the same data (PDF + Excel + CSV)
- Add page numbers, timestamps, and report metadata
- Use streaming for large reports to avoid memory issues
